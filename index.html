Trash to Treasure

This is a proof of concept site which allows a users to search for free or upcycled items in the central Ohio area. Along with the search query comes a dynamically updating weather pane that shows today's weather. To accomplish this we make two API fetch reqeusts, one to the trash nothing API and the openweather API. This was site was designed as part of the OSU coding bootcamp.
